Лабораторная № 1 (1 семестр процедурный подход)
Разработать программу которая получает данные из командной строки и реализует
следуюший функционал:
a) без параметров кратко пишет назначение программы и подсказывает как получить
помощь
b) -h (--help) - выдает подробную справку по программе, с перечислением всех
возможных команд
c) -t (--table) - выдает в табличном виде размеры всех простых типов языка.
d) -с (--calc) арифметическое выражение расчитывает простое арифметическое
выражение вида число операция число (числа целые и вещественные)
e) флаги к команде -с изменяют формат вывода -х (--hex) ответ в 16-ричной формате
-i (--int) входные числа целые (по умолчанию вещественные)
f) если ввод не соответствует командам, то выдать предупреждение и приглашение
воспользоваться справкой для вывода использовать printf();
